Ví Dụ Gọi API T

Endpoint /capture trả về JSON: { ok: true, url: "https://res.cloudinary.com/...", public_id: "..." }. Dùng url để hiển thị ảnh trực tiếp trong web/app của bạn.
JavaScript (Frontend: React/Vue/Vanilla)
const api = "https://tradingviewcapture-721483185057.asia-southeast1.run.app/capture"; const params = new URLSearchParams({ tf: "H1", w: "1440", h: "900", ticker: "OANDA:XAUUSD" }); const res = await fetch(`${api}?${params}`, { mode: "cors" }); if (!res.ok) throw new Error("Capture failed"); const data = await res.json(); if (!data.ok) throw new Error(data.error || "Upload failed"); // Gắn link Cloudinary vào <img id="chart"> document.getElementById("chart").src = data.url;
